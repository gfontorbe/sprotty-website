Open-Source Diagram Visualization Tools
for Web Applications

Sprotty is an open-source framework for developing diagramming applications.
Supported by a community of developers who are passionate about diagramming tools & graphical editors.
Provides a flexible choice of server and client side integrations in Node and Java.

Get started
Learn more

Sprotty provides a framework for rendering diagrams in popular formats, such as SVG, with a flexible choice of layout algorithms, such as the Eclipse Layout Kernel (ELK).

Sprotty provides a set of standard components to make creating common diagram types easy, such as flowcharts, UML diagrams and state machines. Sprotty also provides APIs for further customizing the layout and behavior of your diagrams.

Customizable and Beautiful

Sprotty makes it easy to create beautiful, interactive diagrams in web applications. Since Sprotty is a framework, it leaves you free to customize your diagrams with integration to external tools and services.

From Simple to Advanced

Sprotty can be used for a wide range of diagramming needs, from simple flowcharts to advanced graphical editors that integrate with external services. Sprotty also provides clear ways to apply complex diagram changes to an underlying model, while avoiding complex and error-prone bi-directional transformations.

Less Boilerplate, More Results

Sprotty can be leveraged to quickly produce a working diagram concept, or an advanced diagramming system, without having to be an expert. This lets you focus more on what you want to diagram, rather than how.

At home in any kind of water...

Whether on the server or client, Sprotty has you covered. It provides a Node.js or Java diagram server architecture. It can also be run as a rich-client, without a server. Whichever environment suits your needs, Sprotty can run there too.

Quick to hook in...

Sprotty can be easily integrated with various web frameworks, such as React, Angular and Vue.js and many more. If your choice of framework works with Javascript/Typescript, so will Sprotty.

Sprotty is well suited for client-side diagramming, involving no server-side work whatsoever.

This pairs nicely with external applications, allowing you to provide rich visualizations; such as for DSLs.

First released in 2017, Sprotty has been continuously developed and improved by a community of contributors who firmly believe in Opensource solutions.
